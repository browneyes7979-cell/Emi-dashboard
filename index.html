🔥 2019 여수 NCC 유증기 화재
사고 개요
일시: 2019년 여수 국가산단 NCC(나프타분해시설) 공장
원인: 에틸렌 배관 플랜지 부위 미세 누출 → 풍속 약 2.0m/s 저풍속 조건에서 유증기 체류 → 점화 → 화재
피해: 부상 3명
모델 연관: 현재 풍속 조건과 동일한 메커니즘. 저풍속 + 고온 = 핵심 트리거
이 사례가 모델에 반영된 방식
풍속 3m/s 미만 시 가스 확산 억제 → W_low 변수에 가중치 2.8 부여
해안 염분 + SO₂ → 부식 가속 → H×S 복합항으로 모델화
→ 당시 조건 사후 대입 시 예측 확률 74% (경계 단계)
🔥 2012 여수 세계박람회 직전 폭발
사고 개요
일시: 2012년 여수 세계박람회 개막 직전
원인: 야간 저온 → 주간 급등 온도차 8°C 이상 → 배관 압력 급변 → 안전밸브 오작동 → 폭발
피해: 사망 1명, 부상 4명
모델 연관: ΔT₂₄h(24시간 온도 편차) 5°C 이상 = 핵심 트리거
이 사례가 모델에 반영된 방식
온도 급변 → 배관 열팽창/수축 스트레스 → ΔT₂₄h 변수에 가중치 1.9 부여
→ 당시 조건 사후 대입 시 예측 확률 81% (경계 단계)
🔥 2005 미국 텍사스 BP 정유공장 대폭발
사고 개요
일시: 2005년 3월 23일, 텍사스 시티 BP 정유공장
원인: 이소머화 공정 유체 과충진 + 저풍속 → 유증기 방류탑 주변 체류 → 점화 → 대폭발
피해: 사망 15명, 부상 180명
의미: 저풍속이 유증기 체류를 만든 핵심 악화 인자임을 전세계적으로 입증한 사례
👷 2021 여수 탑상 작업 중 추락 사망
사고 개요
일시: 2021년 여름, 여수 국가산단 내 증류탑 상부 배관 작업
원인: 기온 31°C 환경에서 장시간 작업 → 열피로로 집중력·근력 저하 → 비계 발판 이탈 → 추락
피해: 사망 1명
모델 연관: 기온 25°C 초과 시 T_norm 변수 급상승
이 사례가 모델에 반영된 방식
기온 → 열피로 → 작업 능력 저하 → T_norm 변수에 가중치 1.6 부여
→ 당시 조건(31°C, 비 없음) 사후 대입 시 예측 확률 68% (주의 단계)
👷 산업안전보건공단 고온기 추락 통계
통계 요약
출처: 한국산업안전보건공단 (KOSHA) 2020년 재해통계
핵심 수치: 7~8월 고온기 추락 사고 발생 건수, 비고온기 대비 34% 증가
특이사항: 화학 플랜트 업종에서 특히 두드러짐 (열피로 + 고소작업 복합)
모델 반영: T_norm 변수 선형 관계로 모델화
🏭 2022 여수 산단 VOC 기준 초과 배출
사고 개요
일시: 2022년, 여수 국가산단 A사 화학공장
원인: 고농도 황사 유입 → RTO(축열식 소각로) 필터 급속 폐색 → 차압 급등 → 연소로 우회 → VOC 기준 초과 배출
피해: 환경 법규 위반, 과징금 2억원 부과
모델 연관: PM₂.₅ 농도 50μg/m³ 초과 = 핵심 트리거
이 사례가 모델에 반영된 방식
PM 부하 → 필터 막힘 → 방지효율 저하 → PM_load 변수에 가중치 2.1 부여
대기 역전(기온역전층) 시 배출가스 확산 불량 → Inv 변수 추가
→ 당시 조건 사후 대입 시 예측 확률 63% (주의 단계)
